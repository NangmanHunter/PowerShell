- 모듈 단위	MyModule.psm1	모듈(Module)
- 기능 단위	Get-User.ps1, Set-Config.ps1	함수(Function) 또는 서브 모듈(Submodule)
- 논리적 역할 단위	여러 .ps1이 하나의 기능 그룹 담당	컴포넌트(Component) 혹은 서브유닛(Subunit)
- PowerShell에서는 실제로 .psm1이 “모듈”, 그 안에서 정의된 함수(Function) 들이 개별 기능 단위라고 보는 게 가장 일반적입니다.
- “모듈 단위(.psm1) 안에 여러 기능 스크립트(.ps1)가 존재한다”
- “모듈 안의 개별 함수/컴포넌트 단위(.ps1)”
- 즉, 모듈 = 상위 단위, 함수/스크립트 = 하위 기능 단위라고 표현하는 게 명확합니다.
