- .pssc 파일은 PowerShell의 Desired State Configuration (DSC, 원하는 상태 구성) 기능과 관련된 스크립트 파일입니다. 
- 이는 일반적인 스크립팅보다는 시스템 관리 및 자동화 분야에서 특정 서버나 컴퓨터의 상태를 표준화하고 유지하는 데 사용됩니다.
- Desired State Configuration (DSC)
- DSC는 시스템 관리자가 "이 서버는 최종적으로 이런 상태여야 한다"라고 선언하는 방식의 관리 프레임워크입니다.
- 전통적인 스크립트는 "이렇게 해라(명령형)"고 지시하지만, DSC는 "이 상태를 유지해라(선언형)"고 선언합니다.
- .pssc 파일의 역할
- .pssc 파일은 DSC에서 사용되는 구성(Configuration)을 정의하는 스크립트입니다. 이 파일 안에는 다음과 같은 내용이 포함됩니다.
- 노드(Node): 어떤 컴퓨터(서버)에 이 설정이 적용되어야 하는가?
- 리소스(Resource): 설치해야 할 역할/기능, 설정해야 할 파일 경로, 사용자 계정 등.
- 상태(State): 이 리소스가 존재해야 하는지, 없어야 하는지 등 원하는 최종 상태.
- 요약하자면, .pssc 파일은 "시스템이 어떤 상태여야 한다"는 약속을 담고 있는 설계도 역할을 하는 특수한 PowerShell 스크립트입니다.
